# Copa2026_WebApp
Web App interativo desenvolvido em Python com Streamlit para a cobertura em tempo real da Copa do Mundo de 2026. Arquitetura baseada em dados dinâmicos (JSON) com sistema de links multi-source para transmissões oficiais (CazéTV, Globo, ESPN). Demonstra integração de dados, automação e desenvolvimento ágil para portfólio de TI.
